## Project Description
This is a clean and minimal **Login Page** project designed to practice modern CSS layout techniques. Instead of using traditional "manual" positioning (like negative margins or absolute positioning), this project utilizes **CSS Flexbox** to achieve perfect alignment and responsiveness.

### Key Learnings:
* **CSS Box Model:** Deep dive into how `padding`, `border`, and `margin` interact to create spacing inside and outside of cards.
* **Flexbox Mastery:** Used `display: flex`, `justify-content`, and `align-items` to center the card on the screen and align internal form elements.
* **UI/UX Design:** Implemented hover states, rounded corners (`border-radius`), and a dark-mode background for a modern aesthetic.
* **Clean Code:** Transitioned from manual margin-nudging to a more scalable, gap-based layout.
